Dunder Mifflin API

This REST API manages catalogs and inventory for the Dunder Mifflin Paper Company.

How to access
  • Base URL: http://localhost:8080/api/v1
  • Authentication: HTTP Basic
  • Test users:
    • user / user123 (ROLE_USER)
    • admin / admin123 (ROLE_ADMIN)
What can the API do?
  • List catalogs: GET /api/v1/catalogs (USER, ADMIN) — alias: /api/v1/dundermifflin/catalogos
  • Add product to a catalog: POST /api/v1/catalogs/{catalogId}/products (ADMIN)
  • List inventory: GET /api/v1/inventory (USER, ADMIN) — alias: /api/v1/dundermifflin/inventario
    • Filter by branch: /api/v1/inventory?branchId=b1
    • Pagination: ?page=1&size=20 (max size=100)
  • Update inventory quantity: PATCH /api/v1/inventory/{productId}?branchId={branchId} (ADMIN)
